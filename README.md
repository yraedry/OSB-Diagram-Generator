## Installation

Install my-project with npm

```bash
  pip install -r requirements.txt
  docker build -t osb-crossreferences .
  docker run -i -t CONTAINER-ID /bin/bash
  docker run -i -t c9815ac263df python main.py
```
    
## Deployment

To deploy this project run

```bash
  npm run deploy
```
